# Qpid Messaging API

A connection-oriented messaging API that supports many languages and
platforms.

  || *Languages* || C++, Perl, Python, Ruby, .NET ||
  || *Platforms* || Linux, Windows ||
  || *AMQP versions* || 1.0, 0-10 ||
