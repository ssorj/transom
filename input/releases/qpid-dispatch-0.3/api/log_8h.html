Logging levels

Enumerator
QD_LOG_NONE 

No logging.

High volume messages, o(n) or more for n message transfers.

Debugging messages useful to developers.

QD_LOG_INFO 

Information messages useful to users.

QD_LOG_NOTICE 

Notice of important but non-error events.

QD_LOG_WARNING

QD_LOG_ERROR
