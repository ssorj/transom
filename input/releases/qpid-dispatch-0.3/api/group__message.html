void qd_message_set_message_annotations ( qd_message_t *  msg,
qd_composed_field_t *  da 
)

Set the message annotations for the message. If the message already has message annotations, they will be overwritten/replaced by the new field.

Parameters
msgPointer to a receiver message.
daPointer to a composed field representing the new message annotations of the message. If null, the message will not have a message annotations field. IMPORTANT: The message will not take ownership of the composed field. The caller is responsible for freeing it after this call. Since the contents are copied into the message, it is safe to free the composed field any time after the call to this function.
