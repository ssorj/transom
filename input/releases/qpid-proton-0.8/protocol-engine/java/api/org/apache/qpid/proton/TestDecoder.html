Class TestDecoder

  • java.lang.Object
    • org.apache.qpid.proton.TestDecoder

  • public class TestDecoder
extends java.lang.Object
    TestDecoder is a stripped-down decoder used by interop tests to decode AMQP data. The native version wraps a DecoderImpl, while the JNI version wraps a Data object.
